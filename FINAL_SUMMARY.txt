  🎯 USER FLOWS
═══════════════════════════════════════════════════════════════

1. Registration:
   User → Register (username + email + password)
   → Welcome email sent ✉️
   → Redirect to login

2. Login:
   User → Login (username OR email + password)
   → JWT token received
   → Redirect to dashboard

3. Forgot Password:
   User → Enter email
   → Reset email sent ✉️
   → Click link in email
   → Enter new password
   → Password updated
   → Login with new password

4. URL Shortening:
   User → Login
   → Enter long URL
   → Short URL created
   → View analytics
